# v1.5.1 — open-source polish

**Released:** 2026-05-27 · **Type:** docs + cleanup; no behaviour changes.

This release addresses every finding from the pre-publish audit pass.
It is safe to take — the harness, the agents, the scoring, and the
analysis pipeline are unchanged from `v1.5.0`.

## What changed

### Licensing simplified

- Deleted `NOTICE.md`, `LICENSE-DATA`, `LICENSE.md`.
- Single `LICENSE` (MIT) now covers code, data, charts, and docs prose.
- Citation request lives in the README's bibtex block.

### Documentation rewritten

- **`README.md`** — six-cell headline table, full prereq + quickstart
  with realistic time/cost estimates, "picking a config for real work"
  section distilled from the v1.5 leaderboard, full `arena` CLI table.
- **`AGENTS.md`** — refreshed to v1.5.0 reality: D6 task class
  documented, v1.5 configs added to the tree, conventions reflect that
  single-letter codes (`A`/`B`/`D`) are retired.
- **`CODE_OF_CONDUCT.md`** — short and direct.
- Source-tree docstrings + per-task READMEs — `lib.*` references
  rewritten to `core.*` / `hybrid_arena.*`; "Category D / B / X"
  rewritten to `refactors` / `real-prs` / `puzzles`.

### UX cleanup

- Deleted `scripts/reproduce.sh` — `./arena setup` already does prereq
  checks, and `./arena sweep --config configs/v1.4-smoke.yaml` is the
  smoke command.
- Deleted `logs/v3.3/` — historical sweep logs. `logs/` is now
  gitignored.

### Hygiene

- `src/hybrid_arena/__init__.py` — `__version__` is now real
  (was `"0.1.0"`).
- `pytest` moved from runtime deps to `[dev]` extras.
- Removed unused `pytest -m slow` filter from CI + docs.
- `.github/ISSUE_TEMPLATE/new_model.md` — broken
  `configs/variants/_template.yaml` reference fixed.
- `docs/HYBRID_ROUTING_DESIGN.md` + `docs/release-notes/v1.4.{0,1}.md`
  — `jq` snippets updated from legacy `D::cline::heuristic` to
  `refactors::cline::heuristic`.
- Test aliases `r10_cline`, `r6_mini_swe_agent` → `cline_runner`,
  `mini_swe_runner`.

### Privacy

- Sanitized **263 absolute-path leaks** (`/Users/<owner>/...`) in
  tracked `raw.jsonl` and `progress.log` files. JSON integrity
  re-validated on every row (520 rows, 0 parse errors).

## Reproduce v1.5.0 from v1.5.1

The empirical record is unchanged. To replay the v1.5 hard-task stress
test on your laptop:

```bash
git clone https://github.com/RunanywhereAI/hybrid-arena
cd hybrid-arena
git checkout v1.5.1

python3.12 -m venv .venv && .venv/bin/pip install -e ".[dev,agents]"
cp .env.example .env    # then paste your OPEN_AI_API_KEY
./arena setup

ollama pull gemma4:31b
./arena sweep --config configs/v1.5-hard-gemma4.yaml --strategies always-cloud,heuristic --seeds 42,7,13
./arena analyze results/runs/v1.5-hard-gemma4
```
